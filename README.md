# Customer-Data-Analytics-Project
End-to-end customer behavior analysis project using Python, SQL, and Power BI to extract insights, visualize trends, and support data-driven business decisions.


Dataset Description

The dataset contains 3,900 records and 18 features related to customer transactions.

Key Features:

Customer ID – Unique customer identifier

Age, Gender – Customer demographics

Item Purchased, Category, Size, Color – Product details

Purchase Amount (USD) – Transaction value

Location, Season – Geographic and seasonal context

Review Rating – Customer satisfaction score

Subscription Status – Membership indicator

Shipping Type – Delivery method

Discount Applied, Promo Code Used – Promotional behavior

Previous Purchases – Purchase history

Payment Method – Mode of payment

Frequency of Purchases – Buying frequency

Tools & Technologies

Python (Pandas, NumPy, Matplotlib, Seaborn) – Data cleaning & EDA

SQL (PostgreSQL / MySQL / SQL Server) – Querying & KPI analysis

Power BI – Interactive dashboard development

Gamma – Business presentation creation

Project Workflow
1. Data Cleaning & Preparation

Checked missing values and data consistency

Standardized categorical variables

Converted data types where required

2. Exploratory Data Analysis (EDA)

Analyzed revenue distribution

Studied age and gender purchasing trends

Identified top-performing product categories

Examined seasonal sales behavior

Evaluated impact of discounts and promo codes

3. SQL Analysis

Created structured database tables

Performed aggregations (SUM, AVG, COUNT)

Generated KPIs such as:

Total Revenue

Average Purchase Value

Top Locations by Sales

Most Used Payment Methods

Customer Segmentation

4. Power BI Dashboard

The dashboard includes:

Total Revenue & Sales KPIs

Sales by Category & Season

Customer Demographics Analysis

Payment Method Distribution

Discount & Promo Impact Analysis

Purchase Frequency Insights

Key Insights

Identified high-revenue product categories

Analyzed purchasing behavior across age groups

Measured impact of discounts and subscriptions on revenue

Discovered seasonal sales patterns

Segmented customers based on purchase frequency and spending
